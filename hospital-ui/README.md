# 河口医院智能协同办公平台前端

**英文工程名：hospital-oa-system**

## 开发

```bash
# 进入前端项目目录
cd hospital-ui

# 安装依赖
npm install

# 启动开发环境
npm run dev
```

浏览器访问 `http://localhost:80`

## 发布

```bash
# 构建测试环境
npm run build:stage

# 构建生产环境
npm run build:prod
```

## 说明

该前端工程用于河口医院智能协同办公平台管理端页面开发，当前基于 Vue 2 和 Element UI 构建，并与后端微服务接口联调使用。
